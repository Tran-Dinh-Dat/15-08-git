Creating Tags

Git supports two types of tags: lightweight and annotated.

  • Annotated Tags: git tag -a v1.4 -m "my version 1.4"
  • Lightweight Tags: git tag v1.4-lw
  • (Tagging Later): git tag -a v1.2 9fceb02 (9fceb02: git log --pretty=oneline)

git show v1.2

Sharing Tags

  • git push origin v1.5

Deleting Tags

  • git tag -d <tagname >
  • git tag -d v1.4-lw
  • git push origin --delete <tagname >

Checking out Tags

  • git checkout 2.0.0

Git aliases

  • Git cho phép người dùng tạo ra các aliases làm cho các câu lệnh git trở nên ngắn gọn hơn
  • $ git config --global alias.co checkout
            $ git config --global alias.br branch
            $ git config --global alias.ci commit
            $ git config --global alias.st status

Basic Branching and Merging

  • Basic Branching
    • git checkout -b iss53
    • switched to a new branch "iss53"
    • git commit -a -m 'added a new footer [issue 53]'
    • git checkout -b hotfix
    • switch to a new branch hotfix
    • git commit-a -m 'fixed the broken email address'
    • git checkout master
    • git merge hotfix
